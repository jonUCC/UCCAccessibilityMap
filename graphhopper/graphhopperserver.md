# GraphHopper Local Server

## Requires
Java 21+, Node.js v18+

### Step 1
navigate to UCCAccessibilityMap/graphhopper
Start server in Bash - "java -jar graphhopper-web-11.0.jar server config.yml"
runs graphhopper on localhost:8989

### Step 2
navigate to UCCAccessibilityMap (project root)
in bash, "npm install" (install dependencies - first time only)
"npm run dev" - starts dev server on your computer
http://localhost:5173 will now show our map
